# React Boilerplate
Boilerplate code for a React application.

## Dependencies

 - See [package.json](package.json)

## Project scripts

 - `npm run build`
 - `npm run lint`
 - `npm start`

## Project structure

```
.
├── src
|  ├─ components
|  |  └─ [Component]
|  |     └─ index.js
|  ├─ public
|  |  └─ favicon.ico
|  └─ views
|     └─ index.pug
├── .babrlrc
├── .editorconfig
├── .eslintrc
├── .gitignore
├── karma.conf.js
├── package.json
├── README.md
└── webpack.config.babel.js
```
