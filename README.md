# AdoptOrNot App
React-powered interface for the [Adopt or Not API](https://github.com/mdhgriffiths/adoptornot-api).

## Installation

```
npm install
```

## Dependencies

 - See [package.json](package.json)

## Project scripts

 - `npm run build`
 - `npm run lint`
 - `npm start`
